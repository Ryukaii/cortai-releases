# Cortaí — Releases

Instaladores públicos do Cortaí. O código vive no repositório privado.

## Baixar

Pegue a versão mais recente em **[Releases](../../releases/latest)**.

- **Windows:** `cortai-<versão>-x64.exe`
- **macOS (Apple Silicon):** `cortai-<versão>-arm64.dmg`

Não precisa de conta no GitHub para baixar.
